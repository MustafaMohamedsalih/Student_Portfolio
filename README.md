# HTML & CSS Portfolio Projects

This repository contains all of the projects I created during my web development course.  
Each project demonstrates different aspects of HTML and CSS, from basic layouts to interactive styling.
## Projects

### 1. Portfolio Website
- Built a personal portfolio site using HTML and CSS.
- Customized fonts, colors, and hover effects.
- Added sections: Home, About, GitHub, Projects, and Contact.
- Background image and semi-transparent content boxes for readability.

### 2. Project Showcase Grid
- Created a responsive grid layout to display multiple projects.
- Each project card includes an image, title, and description.
- Added hover effects for interactivity.

### 3. Contact Form
- Designed a styled contact form with name, email, and message fields.
- Added a confirmation message using JavaScript for user feedback.
- Demonstrates form structure and styling.

### 4. GitHub Integration
- Linked portfolio site to my GitHub profile.
- Encourages visitors to explore my repositories and code samples.
